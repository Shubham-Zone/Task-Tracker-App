# Task Tracker App

Task Tracker is a simple Flutter application for tracking tasks.

## Features

- Add tasks with title and description.
- Mark tasks as completed.
- Edit existing tasks.
- Delete tasks.
- View a list of tasks.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/your_username/task_tracker.git
   ```

2. Navigate to the project directory:
   ```
   cd task_tracker
   ```

3. Install dependencies:
   ```
   flutter pub get
   ```

4. Run the app:
   ```
   flutter run
   ```

## Usage

- Add a new task by tapping the "+" button.
- Tap on a task to mark it as completed.
- Long press on a task to edit or delete it.
